Poseidon Vistek — Linux Driver

Bring your COUGAR Poseidon Vistek ARGB cooler's LCD to life on Linux — CPU temperature, power and fan speeds, plus an animated desktop widget.

What it does

🌡️ Live on the LCD

Streams CPU temperature, load, clock, power (W) and fan/pump RPM to the cooler's 1.9" screen.

🪟 Desktop widget

Always-on-top panel mirroring the screen, with two fans that spin at your real RPM.

⚙️ Auto-start

systemd service feeds the panel at 2 Hz and survives reboots; widget auto-starts on login.

🧊 Pump & fan

Reads CPU temp (k10temp), power (RAPL) and fan/pump RPM (nct6687) — no vendor software.

📦 Easy install

One-click .deb for Ubuntu/Debian, or a portable tarball for any distro.

🌈 RGB roadmap

The lighting/color command (0x05) is documented; ARGB control is planned next.

Install

Debian / Ubuntu / Mint / Pop!_OS:

sudo apt install ./vistek-display_1.0.0_all.deb

Any other distribution (tarball):

tar xzf vistek-display-1.0.0.tar.gz
cd vistek-display-1.0.0 && sudo ./install.sh

Then open “Poseidon Vistek Monitor” from your app menu, or run vistek-widget.

How it works

The screen is a USB-HID display (2c65:1000). The driver sends 64-byte command-0x02 reports carrying the sensor values; the cooler's firmware renders them. The panel has a watchdog, so the daemon feeds it continuously at 2 Hz. Full protocol notes are in the repository README. This is an independent, interoperability-focused project — it ships no vendor software or binaries.
